# 🎨 Seaborn Exercise

A comprehensive collection of Jupyter Notebooks exploring **Seaborn** — a Python statistical data visualization library built on Matplotlib. Covers everything from simple plots to advanced multi-panel figures using both built-in and real-world datasets.

## 📌 Topics Covered

| Notebook | Description |
|---|---|
| `Bar.ipynb` | Bar plots with estimators (mean, median, sum) and hue grouping using the Tips dataset |
| `Box Plot.ipynb` | Box plots for spread and outlier detection, single and grouped by category |
| `Cat Plot.ipynb` | Figure-level categorical plots with `catplot()` supporting violin, count, and more |
| `Count Plot.ipynb` | Count plots for categorical frequency analysis with hue — includes real employee data |
| `Heatmap.ipynb` | Heatmaps with annotations on pivot tables and grouped salary data |
| `Hist.ipynb` | Histograms with KDE overlay using Tips and Titanic datasets |
| `KDE Plot.ipynb` | Kernel Density Estimation plots for smooth distribution curves with stacking |
| `Line.ipynb` | Line plots with style mapping across business units and ethnicity groups |
| `Multiple Plot.ipynb` | Multi-panel grids using `FacetGrid` with mapped scatter and bar plots |
| `Pair Plot.ipynb` | Pairwise relationship grids on Tips and Iris datasets with hue and palette options |
| `Relational Plot.ipynb` | Figure-level relational plots with `relplot()` using multi-dimensional encoding |
| `Scatter.ipynb` | Scatter plots with hue, size, and palette — includes real salary vs. age analysis |
| `Strip Plot.ipynb` | Strip plots with dodge and jitter controls for categorical data |
| `Swarm Plot.ipynb` | Swarm plots for non-overlapping categorical distribution display |
| `Violin.ipynb` | Violin plots for distribution shape analysis on Tips and employee bonus data |

## 🗂️ Datasets

- **Built-in Seaborn datasets** — `tips`, `titanic`, `iris`, `glue`
- **`ESD.xlsx`** — Employee dataset with fields: Age, Gender, Department, Ethnicity, Job Title, Annual Salary, Bonus %, Business Unit
- **`expense3.xlsx`** — Expense records with Date, Amount, and Payment Mode

## 🚀 Getting Started

1. **Clone the repository**
```bash
   git clone https://github.com/your-username/Seaborn-Exercise.git
   cd Seaborn-Exercise
```

2. **Install dependencies**
```bash
   pip install seaborn matplotlib pandas numpy openpyxl
```

3. **Launch Jupyter Notebook**
```bash
   jupyter notebook
```

4. Open any `.ipynb` file and run the cells!
